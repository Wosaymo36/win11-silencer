Win11 Silencer
Windows 11 の「おせっかい」な機能や、バックグラウンドで動く不要なプログラムを停止させ、PCを本来の速さに戻すための設定生成ツールです。

⚠️ 重要：実行前の注意
[!CAUTION]
本ツールの実行は、必ず個人所有のPCで行ってください。学校、会社、または他人のPCでは絶対に実行しないでください。

もし組織のPCでどうしても実行したい場合は、必ず管理者やIT部門に問い合わせ、事前に許可を得てください。ただし、重ねて言いますが実行は自己責任となります。本ツールを使用したことによって生じた不具合や損害について、製作者は一切の責任を負いません。

🛠 これは何？
Windows 11 に標準で搭載されている、以下の機能を簡単に無効化するためのレジストリ (.reg) または PowerShell (.ps1) スクリプトを作成します。

AI機能の停止: Recall（画面監視）、Copilot などの無効化

OneDriveの沈黙: 勝手に同期されるドキュメントやピクチャフォルダをローカルに戻す

UIの浄化: 右クリックメニューを旧仕様に戻す、タスクバーの広告やニュースを消す

システム最適化: Edgeのバックグラウンド動作抑制、テレメトリ（診断データ送信）の最小化

🚀 使い方
[Win11 Silencer を開く](あなたのGitHub PagesのURL)

止めたい機能にチェックを入れます（「すべて選択」も可能です）。

**「出力を生成」**ボタンを押し、表示されたコードをコピーします。

A. .reg（レジストリ）形式の場合
メモ帳を開き、コピーしたコードを貼り付けます。

ファイル保存時、ファイル名の末尾を .reg にして保存します（例: fix.reg）。

保存したファイルをダブルクリックして実行し、「はい」を選択してください。

B. .ps1（PowerShell）形式の場合
Windowsのスタートメニューで「powershell」と検索します。

**「Windows PowerShell」を右クリックし、「管理者として実行」**を選択します。

開いた画面で右クリックすると、コピーしたコードが貼り付けられます。そのままEnterキーで実行してください。

※いずれの場合も、実行後にPCを再起動することで設定が反映されます。

🛡 安全性の確認について
本ツールが生成したコードが安全か不安な方、または内容を詳しく知りたい方は、生成されたコードをそのまま Gemini や Claude などの生成AIに貼り付け、「このコードは安全ですか？」と質問してみてください。 AIが各行の意味を解説し、安全性を客観的に判断してくれます。納得したうえでの使用をおすすめします。
