# Win11 Silencer

Windows 11 の「おせっかい」な機能や、バックグラウンドで動く不要なプログラムを停止させ、PCを本来の速さに戻すための設定生成ツールです。

## 🛠 これは何？
Windows 11 に標準で搭載されている、以下の機能を簡単に無効化するためのレジストリ (.reg) または PowerShell (.ps1) スクリプトを作成します。

- **AI機能の停止**: Recall（画面監視）、Copilot などの無効化
- **OneDriveの沈黙**: 勝手に同期されるドキュメントやピクチャフォルダをローカルに戻す
- **UIの浄化**: 右クリックメニューを旧仕様に戻す、タスクバーの広告やニュースを消す
- **システム最適化**: Edgeのバックグラウンド動作抑制、テレメトリ（診断データ送信）の最小化

## 🚀 使い方
1. [Win11 Silencer を開く](あなたのGitHub PagesのURL)
2. 止めたい機能にチェックを入れます（「すべて選択」も可能です）。
3. **「出力を生成」**ボタンを押します。
4. 表示された手順に従って、ファイルを保存し実行してください。
5. **PCを再起動**すると設定が反映されます。

## ⚠️ 注意事項（必ずお読みください）
- このツールは Windows のシステム設定（レジストリ）を書き換えます。
- **実行前に必ず「システムの復元ポイント」を作成しておくことを強く推奨します。**

## 💻 対象ユーザー
- Windows 11 が重くてストレスを感じている方
- 専門知識はないけれど、余計な通信や広告を消したいゲーマーやクリエイターの方
- 勝手に設定を変える OneDrive や Edge の挙動に困っている方
